About

A messaging website built on the SAFE Network with the ability to publish messages and chat in realtime in-browser. The code is designed to readable and showcase some of the features of the SAFE API.

Chaty uses MaterializeCSS and jQuery under the MIT license.

Part of Joseph's SAFE Project. Check out the topic for this project on the SAFE Network forums {here}.
